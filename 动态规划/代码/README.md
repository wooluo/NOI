# 🐍 动态规划代码（Python 版）

> 配套手册：《神奇的动态规划-培训手册.md》
> 适合：刚小学毕业的同学
> 运行方式：把 `.py` 文件下载到电脑，在终端输入 `python3 文件名.py`

## 📁 文件清单

| 文件 | 对应手册章节 | 解决的问题 |
|:---|:---|:---|
| `01_爬楼梯.py` | 第 1 章 | 爬楼梯走法数（斐波那契） |
| `02_最大子段和.py` | 3.1 | 连续选数字让和最大 |
| `03_最长上升子序列.py` | 3.2 | 选出最长的"越来越长"序列 |
| `04_最长公共子序列.py` | 4.1 | 两个序列的最长公共部分 |
| `05_01背包.py` | 4.2 | 每种物品只有 1 件的背包 |
| `06_完全背包.py` | 4.3 | 每种物品无限件的背包 |
| `07_石子合并.py` | 5.1 | 相邻石子合并的最小得分 |

## 🚀 怎么运行

打开"终端"（Terminal），进入这个文件夹，然后：

```bash
python3 01_爬楼梯.py
```

每个文件都可以单独运行，会打印出每一步的计算过程和最终答案。

## 💡 为什么用 Python？

课上的课件用的是 C++ 语言。但 Python **更像人类语言**，读起来轻松，而且**不用编译**，写完就能跑，
非常适合刚入门的同学。等你以后熟练了，再学 C++ 也不迟～ 😉
