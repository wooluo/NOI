## 🚀 怎么用？

### 1️⃣ 读手册（打基础）
用任意 Markdown 阅读器（或 VS Code、Typora）打开：
```
手册/神奇的动态规划-培训手册.md
```
建议每天读 1～2 章，大约 5～7 天读完。

### 2️⃣ 看课件（跟着过一遍）
双击打开（用浏览器 Chrome / Edge / Safari）：
```
课件/神奇的动态规划-课件.html
```
按 **→** 或 **空格** 翻页，按 **Esc** 看所有页面总览。
> ⚠️ 课件首次打开需要联网（加载 reveal.js 框架）。

### 3️⃣ 跑代码（亲手验证）
打开"终端"，进入 `代码/` 文件夹，运行：
```bash
python3 01_爬楼梯.py
```
每个文件都会打印出**每一步的计算过程**和最终答案。

### 4️⃣ 在线代码演示（无需装 Python）⭐
直接双击打开 `代码演示.html`（浏览器）：
- 左边选一个程序，中间看**高亮代码**
- 点 **▶ 运行** 看**模拟运行结果**（打字机式一步步输出）
- 还能 **📋 复制代码** 自己去跑

## 🧠 核心方法论：动规"三步法"

```
① 找状态转移方程（找规律 —— 这是灵魂！）
② 初始化（最开头的几个状态先定好）
③ 按方程一步步算（小的算完，再算大的）
```

**一句话总结：** 把大问题拆成小问题，把小问题的答案记下来，不重复劳动。这就是动态规划的全部秘密！🌟
