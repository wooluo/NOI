# NOI 2026 基础知识练习系统

基于 **《NOI 2026 基础知识题库》**（审订日期 2026-06-15）编制的单页 HTML 在线练习工具，
涵盖笔试全部考点：竞赛环境与规则（Part Ⅰ）、计算机常识与 NOI/IOI 历史（Part Ⅱ）。

正式 NOI 笔试共 50 题（45 单选 + 5 多选），满分 100，题目全部出自题库。

## ✨ 功能特性

- **题型完整**：内置 **104 道题**（91 道单选 + 13 道多选），覆盖题库全部知识点。
- **多种练习方案**：模拟考试（随机 45+5）、全部题目、快速练习、按 Part 分模块练习。
- **两种判题模式**：
  - 🟢 **练习模式**：答题后即时反馈，显示对错与解析。
  - 📝 **考试模式**：交卷后统一批改，模拟真实笔试。
- **随机 / 顺序出题**，可反复练习。
- **计时器**：记录答题用时。
- **成绩与回顾**：得分、正确率、对/错/未答统计；逐题回顾，标出你的答案 vs 正确答案 + 解析。
- **错题重练**：一键只针对本轮错题再练。
- **完全离线**：单文件，无任何外部依赖，双击即可打开。

## 🚀 使用方法

直接用浏览器打开 [`NOI基础知识练习.html`](./NOI基础知识练习.html) 即可，无需安装任何环境。
